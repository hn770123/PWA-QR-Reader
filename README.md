# PWA QRコードリーダー

📷 カメラを使ってQRコードを読み取るプログレッシブWebアプリ（PWA）です。

## 🌟 特徴

- **リアルタイムQR読み取り**: カメラからの映像をリアルタイムでスキャン
- **デュアルカメラ対応**: 前面/背面カメラの切り替えが可能
- **カメラ制御**: カメラのオン/オフを簡単に切り替え
- **PWA対応**: オフラインでも動作し、ホーム画面に追加可能
- **レスポンシブデザイン**: スマートフォンからデスクトップまで対応
- **URLの自動認識**: 読み取ったURLはクリック可能なリンクとして表示

## 🚀 使い方

### オンラインで使用

GitHub Pagesでホストされています:
[https://hn770123.github.io/PWA-QR-Reader/](https://hn770123.github.io/PWA-QR-Reader/)

### ローカルで実行

```bash
# リポジトリをクローン
git clone https://github.com/hn770123/PWA-QR-Reader.git
cd PWA-QR-Reader

# ローカルサーバーを起動（例: Python）
python -m http.server 8000

# ブラウザで開く
# http://localhost:8000
```

### PWAとしてインストール

1. ブラウザでアプリにアクセス
2. アドレスバーの「インストール」ボタンをクリック
3. ホーム画面に追加されます

## 📱 機能説明

### 画面構成

```
┌─────────────────────┐
│   QRコードリーダー    │ ← ヘッダー
├─────────────────────┤
│                     │
│   カメラ映像表示     │ ← 上半分
│   （QRコード検出枠）  │
│                     │
├─────────────────────┤
│ [カメラオン/オフ]    │ ← コントロール
│ [カメラ切替]         │
├─────────────────────┤
│   読み取り結果表示   │ ← 下半分
│                     │
│   [コピー]           │
└─────────────────────┘
```

### ボタン機能

- **📹 カメラオン/オフ**: カメラの起動と停止を切り替え
- **🔄 カメラ切替**: 前面カメラと背面カメラを切り替え
- **📋 コピー**: 読み取った内容をクリップボードにコピー

## 🛠️ 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: レスポンシブデザイン、グラデーション、アニメーション
- **JavaScript (ES6+)**: クラスベースの実装
- **PWA**: Service Worker、Web App Manifest
- **MediaDevices API**: カメラアクセス
- **jsQR**: QRコード解析ライブラリ

## 🌐 ブラウザ対応

- Chrome/Edge (推奨)
- Firefox
- Safari
- Opera

※カメラアクセスにはHTTPS接続が必要です（localhost除く）
